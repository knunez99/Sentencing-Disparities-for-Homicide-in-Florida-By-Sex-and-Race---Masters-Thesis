# Sentencing-Disparities-for-Homicide-in-Florida-By-Sex-and-Race---Masters-Thesis
This project's purpose is to investigate sentencing disparities between sex and race for homicide (and it's charge types) cases in Florida under the CPC sentencing guidelines (1997 to present).  
This repository includes data from the Florida Department of Corrections OBIS System, code that shows the data pipeline, code visualizing the data through summary statistics and statistical models used to investigate the the main question on disparities and presentations on the progress of the project including unused/discarded methods. 
Originally, you would have to execute the DoWork.sh shellscript on your linux/ubuntu that would generate the report, however I just uploaded the generated report for easy viewing. 
The report in question is called mypaper.pdf  -  Just view that to view the final report.
